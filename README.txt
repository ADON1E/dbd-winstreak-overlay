DBD Winstreak Overlay v1.0
Created by: ADONIE

---------------------------------------
DESCRIPTION
---------------------------------------

DBD Winstreak Overlay is a tracker and overlay tool for Dead by Daylight
that allows you to track killer winstreaks and survivor escape streaks.

The app includes a customizable overlay that can be displayed while
playing the game or streaming.

---------------------------------------
FEATURES
---------------------------------------

• Killer winstreak tracking
• Survivor escape streak tracking
    - Solo
    - Duo
    - Trio
    - 4-Man SWF
• Individual streaks and personal bests (PB)
• Portrait selection for solo survivor streaks
• Overlay display for use during gameplay
• Statistics page showing streak performance
• Favorite killers system

---------------------------------------
HOW TO RUN
---------------------------------------

1. Extract the ZIP file.
2. Open the folder.
3. Run:

DBD-Winstreak-Overlay.exe

If Windows shows a security warning:

Click "More Info" → "Run Anyway"

---------------------------------------
IMPORTANT NOTES
---------------------------------------

• Keep all files in the same folder.
• Do not run the program directly from inside the ZIP file.
• The program will automatically create and update:

winstreak_data.json

This file stores your streak progress.

---------------------------------------
KNOWN LIMITATIONS
---------------------------------------

• Windows may display a SmartScreen warning the first time the app runs.
• This is normal for unsigned applications.
• I still have not made this app able to be integrated into OBS but will soon. For now, you just have it on your screen as well.
